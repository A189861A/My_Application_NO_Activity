
#  泛型函数的核心：在函数名前用 <T> 声明类型参数（T 是类型占位符，可自定义名称）
```kotlin
    fun <T> 函数名(参数: T): 返回值类型 {
    // 函数逻辑
    }

// 泛型函数：返回参数的两倍（适配数字类型）
fun <T : Number> doubleValue(value: T): Double {
    // 给泛型加约束：T 必须是 Number 的子类（Int、Double、Float 等）
    return when (value) {
        is Int -> value * 2.0
        is Double -> value * 2.0
        is Float -> value * 2.0
        else -> 0.0
    }
}
```
# 泛型函数 + 扩展函数（Kotlin 特色）
 - 泛型函数常和扩展函数结合，实现对任意类型的扩展
```kotlin
// 泛型扩展函数：将任意类型的元素添加到列表末尾并返回新列表
fun <T> List<T>.addElement(element: T): List<T> {
    val newList = this.toMutableList()
    newList.add(element)
    return newList
}

// 调用示例
fun main() {
    val intList = listOf(1, 2, 3)
    val newIntList = intList.addElement(4) // T=Int → [1,2,3,4]
    
    val strList = listOf("a", "b")
    val newStrList = strList.addElement("c") // T=String → [a,b,c]
    
    println(newIntList)
    println(newStrList)
}
```

#  泛型函数 + 可空类型
```kotlin
// 泛型函数：判断参数是否为空，非空则返回其长度/大小（适配字符串、集合等）
fun <T> getLengthOrSize(value: T?): Int {
    return when (value) {
        is String -> value.length
        is Collection<*> -> value.size
        is Array<*> -> value.size
        null -> 0
        else -> -1 // 不支持的类型返回 -1
    }
}

// 调用示例
fun main() {
    println(getLengthOrSize("Kotlin")) // 6
    println(getLengthOrSize(listOf(1,2,3))) // 3
    println(getLengthOrSize(arrayOf("a", "b"))) // 2
    println(getLengthOrSize(null)) // 0
    println(getLengthOrSize(123)) // -1
}
```
